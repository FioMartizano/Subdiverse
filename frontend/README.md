# Subdiverse

## Project Title
Subdiverse: A Community-Based Web Portal Information System and Real-Time Booking Platform

## Description
Subdiverse is a centralized web-based community portal designed to help subdivisions modernize their services and adapt to the evolving digital landscape. The project focuses on addressing the specific needs of Windward Hills Subdivision, located in Burol Uno, Dasmariñas, Cavite, which serves as the system's sample community for implementation.

The system allows residents to access community announcements, apply for vehicle stickers, reserve subdivision ameneties and parking spaces, participate in community groups and access other HOA-related services.

Subdiverse also provides an administrative system for managing resident accountsm reservations, applications, generate reports, complaints, announcements and other transactions.

## System Requirements
- Node.js
- npm
- Modern Web Browser (e.g. Google Chrome, Microsoft Edge, or Mozilla Firefox)
- Internet Connection

## Installation Steps
1. Clone the repository from GitHub.
2. Open the project in VS Code.
3. Run `npm install`.

## Running the Program
Run:
npm run dev
